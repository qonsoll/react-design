# Theme Provider

For setting responsive web page you need to pass array of breakpoints to ThemeProvider. 

1. Create file breakpoints.js in folder styles/

2. Insert this code to breakpoints.js
```jsx
const breakpoints = [
  '576px',
  '768px',
  '1024px',
  '1280px',
  '1440px',
  '1600px',
  '1920px'
]

export default breakpoints
```

3. Go to index.js file and add import ThemeProvider and pass breakpoint to provider
```jsx
import ReactDOM from 'react-dom'
import { ThemeProvider } from 'styled-components'
import breakpoints from '../styles/breakpoints'

ReactDOM.render(
    <ThemeProvider theme={breakpoints}>
       <App />
    </ThemeProvider>,
    document.getElementById('root')
)
```
